# Contributing to AariyaTech AI Community

Thanks for helping — contributions of any size are welcome.

## How to contribute
1. Read `CODE_OF_CONDUCT.md`
2. Check Discussions → "Welcome & Onboarding"
3. Checkout the `GETTING_STARTED.md` docs
4. Look for issues labelled `good first issue` or `help wanted`
5. Fork the repo, create a branch `feat/your-thing` or `fix/your-thing`, submit a PR

## PR checklist
- [ ] Linked to an Issue (or create one first)
- [ ] Clear description and rationale
- [ ] Small, focused change
- [ ] Update docs if needed
